#7, Ethics and UX.

UX, or User Experience, is a subjective result that is obtained when a person interacts with an object or artifact. There are three main aspects to a UX. First, the object/artifact that is interacted with. Second, the user themselves - their expectations, desires and needs have an influence over the experience. And thirdly, the context in which the user and the object engage with each other.
